从无人机视频到高精度自然驾驶数据

覆盖稳像、检测跟踪、轨迹后处理、车道匹配、标准化输出与场景挖掘的全链路工艺。

STEP 01

视频稳像

基于参考帧的单应性配准，叠加掩码鲁棒估计，使无人机视频帧间像素级对齐，消除漂移与抖动。

STEP 02

目标检测与跟踪

YOLOv8 + BoT-SORT/ByteTrack 多目标跟踪，输出 4 角点 bounding box 与稳定 track_id。

STEP 03

轨迹后处理

滤波、平滑、ID 融合与异常剔除，生成形式化中间轨迹（FML_TRK），保留运动学一致性。

STEP 04

地图匹配

OpenDRIVE 高精地图坐标系下做车道匹配，逐帧确定 lane_id、横向偏移与邻车关系。

STEP 05

标准化输出

对齐 highD 数据集字段：x/y/heading/速度/加速度，自动计算 DHW、THW、TTC 等安全指标。

STEP 06

场景挖掘

从标准化轨迹中自动检索切入、切出、减速等典型交互场景，便于仿真验证与算法回放。

关键模块

稳像模块 step1

  • 多参考帧配准，长时间序列稳定
  • 掩码区域抑制动态目标干扰
  • 稳定后视频与原始坐标系建立单应映射

多目标跟踪 step2

  • YOLOv8 检测器，自定义训练数据增强车辆/行人/两轮车类别
  • BoT-SORT 卡尔曼+外观特征融合，长 ID
  • 每帧输出 4 角点 box + 类别 + 置信度（RAW_TRK.json）

轨迹后处理 step3

  • 跨帧 ID 重连，断裂轨迹补齐
  • 平滑滤波保留运动学连续性
  • 异常帧剔除（出图、检测漂移、ID 抖动）

地图匹配 step4

  • OpenDRIVE 高精地图加载与车道边界采样
  • 逐帧最近车道判定，输出 lane_id 与 lateral offset
  • 同时识别 preceding / following / left/right alongside 邻车

标准化输出 step5

  • highD 兼容字段：x, y, width, height, heading, lon/lat 速度与加速度
  • 安全指标：DHW（车头时距）、THW、TTC
  • 每条轨迹的 META：均速、行驶距离、车道变更次数等

场景挖掘 step6

  • 切入 / 切出 / 减速 / 跟车等典型交互模式自动识别
  • 每个场景保留触发帧、相关 track_id 与上下文窗口
  • 可直接导出仿真回放（CARLA / esmini 等）

精度验证

覆盖位置精度、轨迹平滑度、地图匹配准确率、ID 持久性等关键指标，配合人工标注 ground truth 与 RTK 实测对比。
